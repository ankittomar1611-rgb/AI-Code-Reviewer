\# AI Code Reviewer 🚀



An AI-powered Code Reviewer built using Python and Flask.



\## Features



\- Review Python code

\- AI-generated suggestions

\- Clean UI

\- Flask backend

\- Easy to use



\## Technologies Used



\- Python

\- Flask

\- HTML

\- CSS

\- Gemini API



\## Installation



Clone the repository:



git clone https://github.com/ankittomar1611-rgb/AI-Code-Reviewer.git



Go to project folder:



cd AI-Code-Reviewer



Create virtual environment:



python -m venv venv



Activate virtual environment:



venv\\Scripts\\activate



Install dependencies:



pip install -r requirements.txt



Run the project:



python app.py
